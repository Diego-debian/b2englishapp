# T11.8: Feed Unified Palette

> Feature flag: `NEXT_PUBLIC_FEATURE_UI_V11`

## Goal

Ensure `/content/feed` shows a **light, professional theme** when flag is OFF (not dark), and shows the same base palette with polish when flag is ON.

---

## OFF/ON Checklist

### Flag OFF (`NEXT_PUBLIC_FEATURE_UI_V11=false`)
- [ ] Page background: soft stone-50 (not dark slate-950)
- [ ] Cards: white with stone-200 borders
- [ ] Title: stone-900 (dark, readable)
- [ ] Body text: stone-600 (neutral)
- [ ] Meta/dates: stone-500 (subtle but visible)
- [ ] Tags: stone-100 background
- [ ] Links: violet-600
- [ ] No dark mode appearance

### Flag ON (`NEXT_PUBLIC_FEATURE_UI_V11=true`)
- [ ] Same light base palette
- [ ] Cards use ds.card.root polish (shadow, hover effects)
- [ ] Typography uses ds.typo.* tokens
- [ ] Forms use ds.form.input tokens
- [ ] Softer focus rings (violet-400/50)

---
