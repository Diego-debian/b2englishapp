# T1.1 Estrategia de Visibilidad de Nivel

## Problema
El "Nivel" del usuario se calcula globalmente basado en XP total. Sin embargo, en el modo **Focus**, el XP ganado es 0 (por diseño). Mostrar el Nivel o Barra de XP en este contexto es confuso y da una falsa sensación de progreso acumulado durante la sesión.

## Reglas de Visibilidad

### 1. Focus Mode
- **Ruta**: `/practice/focus` (y subrutas).
- **Acción**: **OCULTAR COMPLETAMENTE** el badge de Level y la barra de XP del Header.
- **Razón**: El modo Focus es "sin distracciones" y "sin XP". La presencia de gamificación global contradice el propósito.

### 2. Global (Dashboard, Verb/Tense lists)
- **Ruta**: Cualquier otra.
- **Acción**: Mostrar Level y XP normalmente.

## Implementación Técnica
- **Componente**: `frontend/components/GamifiedHeader.tsx`
- **Lógica**: Detectar si la ruta actual comienza con `/practice/focus`. En ese caso, suprimir el renderizado del badge de Nivel y la barra de XP en el encabezado. El resto de elementos (Logo, Navegación, Login/Logout) deben permanecer visibles.

## Verificación
- Navegar a `/practice/focus`: Header debe estar limpio de gamificación (Badge y XP ocultos).
- Navegar a `/dashboard`: Header debe mostrar Level y XP normalmente.
